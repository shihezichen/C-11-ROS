### Protobuf
1. 简介:
   
   ```shell
   protobuf 全称 Protocol Buffers, 简称GPB/PB, 由Google发明, 是QQ等IM采用的协议. 比JSON/XML等传输效率都要高(比JSON高10倍以上) .

   官网: https://developers.google.com/protocol-buffers
   github网址: https://github.com/protocolbuffers/protobuf
   
   编译器:  https://github.com/protocolbuffers/protobuf/releases/download/v3.11.4/protoc-3.11.4-linux-x86_64.zip
   ```
   
2. 准备工作

   ```shell
   # 下载Protobuf编译器
   https://github.com/protocolbuffers/protobuf/release
   选择 C++ 版本
   $ wget https://github.com/protocolbuffers/protobuf/releases/download/v3.11.4/protoc-3.11.4-linux-x86_64.zip
   
   # download 
   $ wget https://github.com/bazelbuild/bazel/releases/download/3.1.0/bazel-3.1.0-installer-linux-x86_64.sh
   
   
   
   ```
